Prerequisite
1. Download Visual Studio 2022 through https://visualstudio.microsoft.com/downloads/, community edition is okay for this
2. Install .net 8 through Visual Studio Installer

How to run the project?
1. Open the solution via Visual Studio 2022
2. Run the project "Quote"
3. It will automatically bring you to isalive page that means you run it successfully
4. You could find the API through the swagger https://localhost:59251/swagger/index.html or the provided file swagger.json
5. There is an example of isAlive API consuming in Reqnroll project, you should be able to run it and the scenario will pass
6. You are good to start with your acceptance criteria for creating the quote service
